\ssr{ЗАКЛЮЧЕНИЕ}

В рамках данной лабораторной работы была успешно достигнута основная цель.

Все поставленные задачи выполнены в полном объёме:
\begin{itemize}[label=---]
	\item разработаны рекурсивный и нерекурсивный алгоритмы решения задачи;
	\item описаны средства разработки и инструменты замера процессорного времени выполнения реализации алгоритмов;
	\item реализованы разработанные алгоритмы на языке программирования Rust;
	\item выполнено тестирование реализации алгоритмов на корректность работы;
	\item выполнена теоретическая оценка затрачиваемой памяти для каждого алгоритма, включая анализ высоты дерева рекурсивных вызовов. Оценка затрат памяти выявила, что нерекурсивный алгоритм требует больше памяти;
	\item оценена трудоёмкость алгоритмов в худшем случае. Оценка выявила, что рекурсивный алгоритм более трудоемкий в худшем случае;
	\item выполнено сравнение результатов замеров процессорного времени. Оценка выявила, что рекурсивный алгоритм работает медленнее нерекурсивного.
\end{itemize}
