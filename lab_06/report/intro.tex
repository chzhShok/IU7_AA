\ssr{ВВЕДЕНИЕ}

Цель работы: сравнительный анализ метода полного перебора и метода на основе муравьиного алгоритма для поиска кратчайшего незамкнутого пути в неориентированном взвешенном графе.

Для достижения поставленной цели необходимо решить следующие задачи:

\begin{itemize} 
	\item описать метод полного перебора и метод на основе муравьиного алгоритма;
	\item реализовать два метода;
	\item выполнить оценку трудоёмкости реализованных алгоритмов по разработанным схемам;
	\item выполнить параметризацию алгоритма поиска минимального пути на основе муравьиного алгоритма;
	\item провести сравнительный анализ двух рассмотренных методов решения задачи коммивояжёра;
	\item по итогам исследования дать рекомендации о значениях параметров.
\end{itemize}
